systemd を使用したサービス管理について学習します。このモジュールでは、ステータスと検証、開始、停止、再起動、起動時のサービス有効化、ダウンタイムなしでの設定リロード、journalctl を使用したサービスログの表示について扱います。実践的なラボと実世界の課題を通じて、これらの必須の Linux スキルを習得します。
